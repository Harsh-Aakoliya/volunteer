# APK Distribution Folder

This folder contains APK files for version 1.0.1.

## How to use:

1. Place your APK file in this folder
2. The file should be named with a `.apk` extension
3. The system will automatically detect and serve the APK file

## Example:
- `app-1.0.1.apk`
- `volunteer-app-1.0.1.apk`
- Any other name with `.apk` extension

## Note:
- Only one APK file should be present in this folder
- The system will serve the first APK file it finds
- Make sure the APK file is properly signed and ready for installation
